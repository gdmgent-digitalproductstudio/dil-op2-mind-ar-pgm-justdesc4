# Mind AR - Meet the Animal

Mind AR – Meet the Animal is een kleine experimentele webapplicatie waarin AR wordt gebruikt om dieren tot leven te brengen. Het project is bedoeld om MindAR te verkennen en te demonstreren hoe gezichtsherkenning en image tracking gecombineerd kunnen worden met 3D-modellen en geluid.

# Project link

[https://gdmgent-digitalproductstudio.github.io/dil-op2-mind-ar-pgm-justdesc4/index.html](https://gdmgent-digitalproductstudio.github.io/dil-op2-mind-ar-pgm-justdesc4/index.html)

# Gebruikte technologieën

- MindAR.js

- A-Frame

- HTML / CSS / JavaScript

- 3D-modellen (GLTF/GLB) , sketchfab

- Audiobestanden (MP3)
